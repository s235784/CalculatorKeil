# STM32Calculator
 一个用STM32F407和串口屏制作的简单计算器，能够进行四则运算、三角函数运算、指对数运算，且有正确的计算优先级。
 
 
 ## 硬件
 单片机：STM32F407VET6
 
 串口屏：淘晶驰 TJC4827X343_011R
 
 ## 连接方式
 串口屏RX接单片机PA9/TXD1
 
 串口屏TX接单片机PA10/RXD1
